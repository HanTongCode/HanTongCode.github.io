选择器

  • 基础选择器由单个选择器组成

    • 标签选择器
      能快速为页面中同类型的标签设置样式
      不能设计差异化样式
    • 类选择器
      样式点定义，结构类调用
      多类名使用空格隔开
      可以多次调用
    • id 选择器
      样式#定义，结构 id 调用
      只能调用一次，一般用于页面唯一性元素
    • 通配符选择器
      选中页面中所有的元素

  • 复合选择器，由两个或多个基础选择器组成

    • 后代选择器（重要）
      外层标签写在前面，内层标签写在后面，中间用空格分隔
    • 子选择器
      只会选择子
      元素 1> 元素 2
    • 并集选择器（重要）
      可以选择多个标签
      元素 1，元素 2
    • 伪类选择器
      用于向某些选择器添加特殊的效果或选择第一个，第 n 个元素

字体属性

  • 字体系列 font-family
  • 字体大小 font-size
    谷歌浏览器默认字体 16px
  • 字体粗细 font-weight
    normal 正常字体，相当于 400（后面没有单位）
    bold 粗体，相当于 700
  • 字体样式 font-style
    normal 正常字体样式
    italic 斜体
  • 复合属性
    font：font-style font-weight font-size font-family （不能变换顺序）
    其中 font-szie 和 font-family 不能省略
  • 标题类字体大小比较特殊，需要单独指定大小

文本属性

  • 文本的颜色 color
    预定义的颜色值
    十六进制
    rgb 代码
  • 对齐文本 text-align
    设置元素内文本内容的水平对齐方式
    left，right，center
  • 装饰文本 text-decoration
    可以给文本添加下划线、删除线、上划线等
    none 没有装饰线
    underline 下划线
    overline 上划线
    line-through 删除线
  • 文本缩进 text-indent
    段落的首行缩进
    em 是一个相对单位，就是当前元素 font-size 一个文字的大小。如果当前元素没有设置文字，则会按照父元素的 1 文字大小
  • 行间距 line-hight
    控制文字行与行之间的距离
    行间距包括上间距，文本高度，下间距

元素显示转换

  • display：block 转换为块级元素
  • display：inline 转换为行内元素,转换之后宽度和高度设置无效
  • display：inline-block 转换为行内块元素

垂直居中

  • 方式一
    line-height = height

背景属性

    可以同时添加背景颜色和图片，图片会压在上面
  • 背景颜色
    background-color
    默认值为 transparent
  • 背景图片
    background-image:url()
  • 背景平铺
    background-repeat
    no-repeat
    repeat,默认
    repeat-x
    repeat-y
  • 背景图片位置
    background-position -参数是方位名词
    如果指定的两个值都是方位名词，则两个值的前后顺序无关
    如果只有一个方位名词，则第二个参数默认为 center -参数是精确单位
    顺序不可变
  • 背景图像固定
    background-attachment
    设置是否固定或者随页面的其余部分滚动
    scroll 滚动
    fixed 固定
  • 背景颜色半透明
    background:rgba(0,0,0,0.3)

三大特性

  • 层叠性
    样式冲突，就近原则
  • 继承性
    子标签会继承父标签的某些样式
    text-
    line-
    font-
    color
  • 优先级
    无穷大： ！important
    1，0，0，0：行内样式
    0，1，0，0：id 选择器
    0，0，1，0：类选择器/伪类选择器
    0，0，0，1：元素选择器
    0，0，0，0：继承/*
    a 标签有默认的样式

盒子模型

  • border
    • 边框宽度
      border-width
    • 边框样式
      border-style
      none:无边框
      hidden：隐藏边框
      dotted：点线
      dashed：虚线
      solid：实线
    • 边框颜色
      border-color
  • 内容
    • content
    • padding-left/right/top/bottom
      padding 会撑开盒子（宽度/高度减去 padding）
      一个值：上下左右
      两个值：上下，左右
      三个值：上，左右，下
      四个值：上，右，下，左
      如果盒子本身没有指定 width/height 属性，则此时 padding 不会撑开
  • margin 外边距

阴影

  • 盒子阴影
    box-shadow
    -h-shadow,必须，水平阴影的位置
    -v-shadow，必须，垂直阴影的位置
    -blur，模糊距离
    -spread，阴影尺寸
    -color，阴影颜色
    -inset，外部阴影改为内部阴影
  • 文字阴影
    text-shadow
    -h-shadow,必须，水平阴影的位置
    -v-shadow，必须，垂直阴影的位置
    -blur，模糊距离
    -color，阴影颜色

布局

  • 普通流
  • 浮动
    可以让多个块级元素一行内排列显示
    属性：float，none，left，right
    特性：
    浮动元素会脱离标准流（不会保留原来的位置）
    浮动的元素会一行内显示并且元素顶部对齐
    浮动的元素会具有行内块元素的特性(重点)
    添加浮动后，大小由内容决定
  • 定位
